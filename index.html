SpamShield: Machine Learning-Based Email Spam Detection Using Decision Trees

This project aims to design and develop an intelligent spam email detection system using machine learning and natural language processing (NLP) techniques, enabling the accurate classification of emails as spam or legitimate.

Spam Shield is an intelligent email spam detection system designed to accurately classify emails as spam or legitimate using advanced machine learning algorithms like Support Vector Machine (SVM), Naive Bayes, and Random Forest. By leveraging Natural Language Processing (NLP) techniques, the system analyzes the content and patterns within emails to identify suspicious or harmful messages. Spam Shield aims to enhance email security by reducing unwanted emails and protecting users from potential threats.

Python Support Vector Machine (SVM) TensorFlow/Keras K-Nearest Neighbors (KNN)
